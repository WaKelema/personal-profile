# Personal Profile Website

This is my personal portfolio website created using **HTML, CSS, Bootstrap, and GitHub Pages**.  
The website provides information about me, my skills, my education, and how to contact me.

## 🌐 Live Website

You can view the website here:

https://wakelema.github.io/personal-profile/

## 📂 Project Structure


## 🧑‍💻 Pages Included

### Home (index.html)
The home page introduces the website and includes a hero section with a short description about me.

### About (about.html)
This page contains background information about me, my interests, and my passion for technology.

### Skills (skills.html)
This page lists the technical and personal skills I am currently developing, including programming and web development.

### Education (education.html)
This section highlights my academic background and the subjects I am studying.

### Contact (contact.html)
This page provides ways to contact me including email and GitHub.

## 🛠 Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- GitHub Pages

## 🎯 Purpose of the Project

The purpose of this project is to:

- Practice web development skills
- Learn how to build multi-page websites
- Understand how to host websites using GitHub Pages
- Create a personal online profile
